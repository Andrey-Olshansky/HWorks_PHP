# Алгоритм развертывания PHP-приложения на InfinityFree

## 1. Регистрация и создание аккаунта хостинга

1.1. Перейдите на сайт https://infinityfree.net
1.2. Нажмите "Sign Up Free" → выберите вход через Google
1.3. После авторизации нажмите "Create Free Account"
1.4. Выберите "Use a subdomain"
1.5. Введите имя проекта (например: php-homework)
1.6. Выберите доменную зону .rf.gd
1.7. Нажмите "Create Account"

## 2. Доступ к файловой системе

2.1. В личном кабинете найдите созданный аккаунт
2.2. Нажмите кнопку "Manage" напротив него
2.3. В открывшейся панели нажмите "Login to cPanel"
2.4. В cPanel найдите и откройте "File Manager"

## 3. Подготовка структуры файлов для Д/З

3.1. В File Manager перейдите в папку `public_html`
3.2. Удалите стандартный index.php если он присутствует
3.3. Создайте необходимые файлы через "+File":

Структура файлов:
- index.php        (главная страница с формой)
- post.php         (обработчик формы)
- exit.php         (выход из сессии)
- style.css        (опционально, для стилей)

3.4. Скопируйте код из вашего репозитория GitHub

## 4. Настройка сессий PHP

4.1. Убедитесь, что в коде используется `session_start()`
4.2. Проверьте что все пути к файлам относительные
4.3. Важно: на хостинге автоматически поддерживаются сессии

## 5. Тестирование приложения

5.1. Откройте в браузере ваш адрес: ваш-проект.rf.gd
5.2. Проверьте:
   - Отображение формы при первом посещении
   - Сохранение имени после отправки формы
   - Приветствие без формы при активной сессии
   - Выход по ссылке и очистка сессии

## 6. Решение частых проблем

6.1. Если сессия не сохраняется:
   - Проверьте `session_start()` в начале каждого файла
   - Убедитесь что нет вывода до `session_start()`

6.2. Если страница не открывается:
   - Проверьте наличие index.php в public_html
   - Убедитесь что аккаунт активирован (Status: Active)

6.3. Ошибки подключения:
   - Всегда используйте `localhost` для подключения к БД
   - Проверьте права на файлы (рекомендуется 644)

## 7. Публикация ссылок для Д/З

## 8. Дополнительные возможности InfinityFree

8.1. Для работы с БД: используйте MySQL Databases в cPanel
8.2. Для импорта/экспорта БД: используйте phpMyAdmin
8.3. Для загрузки больших проектов: используйте FTP
    - Данные FTP: хост=ваш-сайт.rf.gd, порт=21
    - Логин/пароль создаются в разделе FTP Accounts

## 9. Проверка перед сдачей Д/З

- [ ] Все файлы загружены в public_html
- [ ] Приложение открывается по ссылке
- [ ] Форма отправляет данные корректно
- [ ] Сессия сохраняется между страницами
- [ ] Кнопка выхода очищает сессию
- [ ] Репозиторий на GitHub содержит весь код

## 10. Как сдать Д/З:

1. Загрузите код на InfinityFree (по алгоритму выше)
2. Убедитесь что приложение работает по ссылке ваш-проект.rf.gd
3. Отправьте преподавателю:
	3.1. Ссылку на GitHub репозиторий
	3.2. Ссылку на работающее приложение
